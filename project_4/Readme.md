# Retail Sales Analysis

## Project Overview
This project analyzes retail sales data to identify sales trends, profitable categories, regional performance, and business insights using Excel, MySQL, and Power BI.

## Tools Used
- Excel
- MySQL
- Power BI

## Key Tasks Performed
- Cleaned and transformed raw sales data
- Analyzed sales, profit, quantity, and regional performance using SQL
- Created interactive Power BI dashboard with KPIs and slicers
- Generated business insights and recommendations

## Dashboard KPIs
- Total Sales
- Total Profit
- Total Quantity Sold
- Profit Margin %

## Key Insights
- Home Goods category generated the highest quantity sold
- West region contributed the highest profit
- December recorded the highest sales growth
- Some categories had high sales but low profit margins
- Sales declined significantly during February

## Business Recommendations
- Increase inventory for high-demand categories
- Improve pricing strategy for low-margin products
- Focus marketing efforts in high-performing regions
- Prepare seasonal campaigns before December sales spikes
